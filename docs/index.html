Kotodama

A desktop language-learning application that turns reading into your vocabulary trainer.

Import texts, read naturally, and track every word and phrase you encounter — from first sight to well known. Built with Qt for macOS, Windows, and Linux.

Features

Read Real Texts Import plain text and read it in a clean, focused viewer.
Phrase-Aware Matching Recognizes multi-word phrases, not just isolated dictionary words. A trie-based engine matches the longest known phrase at any position.
Five Learning Levels Mark terms as Recognized, Learning, Known, Well Known, or Ignored. Each level gets its own highlight color.
Inline Editing Hover or use arrow keys to focus a word, press Space to edit its pronunciation, definition, and level without leaving the reader.
Keyboard Navigation Navigate tokens with arrow keys. While reviewing a term, press 1–5 to set its level instantly.
Vocabulary Import Bring in existing word lists from CSV files.
Persistent Progress Your texts, terms, and metadata are stored locally in SQLite. Nothing leaves your machine unless you configure it to.
Cross-Platform Native builds for macOS, Windows, and Linux.
